# fomo
Fear Of Missing mnemOnics

This code will collect the JWST Calibrated Engineering Data also called mnemOnics in order to correct your light curves :) 
https://mast.stsci.edu/portal/Mashup/Clients/jwstedb/jwstedb.html


## Installation

(basically just that you have the dependancies)
create a new directory and git clone the github there:

git clone https://github.com/sebastian-zieba/fomo.git

conda create -n fomo python==3.13.0
conda activate fomo
cd Desktop/Projects/Open_Source/fomo/fomo
pip install -e .
